## 优化内容

### 1. CSS背景图懒加载
- 添加 `.hero-loaded` 类，初始不加载背景图
- 元素进入视口时添加类，触发背景图加载

**修改文件**: `assets/css/style.css`
```css
.hero {
  background-image: none;
}
.hero.hero-loaded {
  background-image: linear-gradient(...), url('../images/hero-bg.png');
}
```

### 2. HTML懒加载属性
- 给hero section添加 `data-bg="hero"` 属性

**修改文件**: `index.html`, `index_en.html`
```html
<section class="hero" data-bg="hero">
```

### 3. JS懒加载逻辑
- 使用 IntersectionObserver 观察hero元素
- 进入视口时添加 `hero-loaded` 类

**修改文件**: `assets/js/main.js`, `assets/js/main_en.js`
```js
var hero = document.querySelector('[data-bg="hero"]');
if (hero && 'IntersectionObserver' in window) {
  var obs = new IntersectionObserver(...);
  obs.observe(hero);
}
```

### 4. og-image.png
- 复制 hero-bg.png 为 og-image.png
- 更新 og:image meta标签指向 og-image.png

**新文件**: `assets/images/og-image.png`
